# 구현 상태

이 문서는 현재 main을 기준으로 실제 동작하는 범위와 남은 범위를 구분합니다.

## DB와 기준 데이터

- PostgreSQL 18.4 Docker Compose 실행 환경
- Flyway 7개 마이그레이션과 Testcontainers 빈 DB 검증
- 현재 2026-1 강의 카탈로그
- 2020~2026 과거 강의 개설 이력
- 2016~2026 교육과정 필수과목
- 2020~2026 학과·학번별 졸업 학점·교양요건
- 2026 학과별 졸업인증과 보존된 구 졸업요건
- 정규 학과·전공 코드, 연도별 별칭, 학사 데이터 FK 연결
- 공개 저장소에서 제외된 SQL 페이로드의 체크섬·멱등 적재 스크립트

## 구현 및 통합 테스트 완료

### 인증·보안

- 이메일 OTP 요청·검증
- 로그인 세션 조회·유지와 로그아웃
- 회원 탈퇴 시 사용자 소유 데이터 전체 삭제와 세션 무효화
- Spring Security 세션 인증
- OTP 만료·재전송 대기·시도 횟수 제한·BCrypt 저장
- CSRF, CORS, 보안 쿠키 설정

현재 구현은 이메일 OTP 방식입니다. 소셜 로그인은 아직 구현되지 않았습니다.

### 학사·강의·리뷰

- 학과·학기·데이터 버전 조회
- 강의·분반 조회, 과목·교수 검색, 학과·분류·학점·요일 필터
- 이름·리뷰 수·베이지안 평점·인기순 정렬
- 공개 리뷰 목록과 과목·교수별 조회
- 인증 사용자의 리뷰 작성·내 리뷰 조회·수정·삭제
- 작성자 소유권, 중복 리뷰, 별점·교수 검증

강의 원본에 수강 대상 학년 필드가 없어 학년 필터는 제공하지 않습니다.

### 이수과목·졸업요건

- 이수과목 직접 등록·조회·수정·삭제
- 상태·학기 필터와 학점 요약
- 내 시간표의 과목을 수강 중 이수과목으로 가져오기
- 수강 중 과목의 이수 완료 전환
- 입학연도·학과·학생 구분·전공 방식별 졸업요건 조회
- 이수학점·교양 영역·필수과목 판정
- 부족 학점·과목 안내와 현재 학기 후보 추천
- 공식 근거·정규화 경고·자동 판정 불가 항목 반환

### 시간표·자동 편성

- 사용자 소유 시간표 CRUD와 분반 추가·삭제
- 분반 시간 충돌 검사
- OR-Tools CP-SAT 기반 자동 편성 작업 생성·조회·취소
- 사용자 소유권과 학사 DB 기준 분반 검증

### API 문서

- 도메인별 Markdown API 명세와 프론트 연동 안내
- OpenAPI JSON·YAML 자동 생성
- 루트 주소에서 열리는 Scalar API 문서와 기존 Swagger UI
- 세션 쿠키와 CSRF 요구사항의 OpenAPI 표시

## 아직 구현되지 않음

- 소셜 로그인과 공급자별 계정 연결 정책
- 졸업판정에 필요한 입학연도·학생 구분·전공 방식 입력 API
  - 현재 내 정보 수정은 이름·학년·학과만 지원
- 시간표 즐겨찾기·복사·이전 학기·공유 링크
- 다른 분반 교체와 대체 분반 추천
- OCR 업로드·인식·보정·원본 삭제
- 프론트 최종 계약용 단일 성공 응답·오류 응답 형식

상담 횟수, 개인별 학점 인정, 자격 증빙 유효성, 예외 승인, 학교 포털의 최종
졸업사정은 현재 데이터만으로 자동 확인할 수 없습니다.

## 공개 저장소에 포함하지 않는 항목

- 실제 사용자 계정·개인정보·세션
- OTP 원문과 운영 비밀값
- 교수명과 원천 payload를 포함할 수 있는 기준 데이터 SQL 페이로드
- 운영 DB 백업
