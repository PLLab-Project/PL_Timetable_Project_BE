Flyway V2026.07.20.005 적용 후 PostgreSQL 시스템 카탈로그에서 추출한 물리 스키마입니다. 모든 애플리케이션 테이블의 컬럼, 키, 외래키, CHECK·UNIQUE 제약과 인덱스를 한 파일에서 확인할 수 있습니다.

기준: backend/src/main/resources/db/migration의 마이그레이션 5개. Flyway 내부 관리 테이블 flyway_schema_history만 ERD에서 제외했습니다.
